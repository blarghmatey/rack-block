rack-block by udzura

A rack middleware for handling search bot access, ip block, etc.

Dependencies

rack (>= 1.3)

Install

gem install rack-block

then edit config.ru:

use Rack::Block do
  bot_access do
    path '/foo' do
      halt 404
    end
  end
end
run YourApp.new
